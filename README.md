# Digital-Clock


A modern and responsive Digital Clock built using HTML, CSS, and JavaScript that displays real-time hours, minutes, and seconds with a clean user interface. This project updates the time every second using JavaScript’s setInterval() function, providing a smooth live clock experience. Designed with stylish gradients, glowing effects, and a centered layout to improve both functionality and visual appeal.

✨ Features:

Live real-time clock updates Responsive and modern UI design Smooth and clean layout Built with pure HTML, CSS & JavaScript Beginner-friendly project for practicing DOM manipulation and timing functions

🚀 This project helped me improve my JavaScript skills, especially working with Date() objects, DOM updates, and dynamic UI styling.
